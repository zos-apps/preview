## About

View and annotate images and PDF documents.

## Features

- Image viewing
- PDF support
- Annotations
- Markup tools
- Export

## Installation

### From App Store
Open the **App Store** in zOS and search for "Preview".

### Via npm
```bash
npm install @zos-apps/preview
```

## Usage

- Open files to view
- Use markup toolbar
- Export in various formats



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
